# Planning Directory

This directory contains design documents, interview notes, and templates used during the planning phase — before execution begins.

## Files

| File | Purpose |
|------|---------|
| `sdd_template.md` | Blank SDD template — populated by `skills/sdd.md` or `skills/grill_me.md` |
| `sdd.md` | Your project's completed SDD (generated — do not create manually) |
| `phase_spec_template.md` | Blank phase spec template — copy to `future_phases/` |
| `notes.md` | Raw brainstorm / interview notes (optional input to `skills/sdd.md`) |
| `retro_*.md` | Per-phase retrospectives (generated by `skills/retro.md`) |

## Workflow

```
planning/notes.md (optional)
        ↓
skills/grill_me.md  →  planning/sdd.md
                              ↓
                  skills/phase_spec_writer.md  →  future_phases/01_*.md
                                                   future_phases/02_*.md
                                                   ...
```
